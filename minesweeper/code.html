2️⃣ GameEngine – floodReveal (DFS Recursion)▼
GameEngine.java ב-domain/usecase — Pure Java, ללא Android SDK.
⚠️ לבגרות: זהו DFS — הפונקציה קוראת לעצמה. לא BFS עם Queue!
// חשיפת שטחים ריקים — DFS Recursion
private void floodReveal(int x, int y) {
    for (int dy = -1; dy <= 1; dy++) {
        for (int dx = -1; dx <= 1; dx++) {
            if (dx == 0 && dy == 0) continue;
            int nx = x + dx, ny = y + dy;
            if (nx >= 0 && nx < width && ny >= 0 && ny < height) {
                Cell c = board[ny][nx];
                // isCovered() = visited guard — תא REVEALED לא יעובד שוב
                if (c.isCovered() && !c.isMine()) {
                    c.setState(CellState.REVEALED);
                    game.incrementTilesRevealed();
                    // רק תאים ריקים (0 מוקשים שכנים) קוראים ל-recursion
                    if (c.getAdjacentMines() == 0) {
                        floodReveal(nx, ny); // ← DFS: קריאה רקורסיבית
                    }
                }
            }
        }
    }
}
🎯 נקודות מפתח:
• DFS = הפונקציה קוראת לעצמה (recursion), לא BFS עם Queue
• isCovered() = visited guard — לאחר setState(REVEALED), התא כבר לא covered
• בדיקת 8 שכנים עם לולאות dx/dy ∈ {-1,0,1}
• recursion רק אם adjacentMines == 0 (תא ריק לחלוטין)

public class GameViewModel extends ViewModel {
    private final GameRepository repository;
    private final MutableLiveData<GameStatus> gameState = new MutableLiveData<>();
    private final MutableLiveData<Game> currentGame    = new MutableLiveData<>();
    private final MutableLiveData<Integer> timer       = new MutableLiveData<>(0);

    public void finishGame(boolean isWin, long timeSeconds, String difficulty) {
        GameResult result = new GameResult(difficulty, timeSeconds, isWin);
        // 1. עדכון UI מיידי (main thread → setValue OK)
        gameState.setValue(isWin ? GameStatus.WON : GameStatus.LOST);
        // 2. שמירה א-סינכרונית — לא חוסמת UI
        repository.saveGameResult(result);
        // 3. חישוב XP ב-background
        XpManager.calculateAndAddXp(isWin, difficulty);
    }

    @Override
    protected void onCleared() {
        super.onCleared();
        if (timer != null) timer.cancel(); // מניע memory leak
    }
}
חוק: finishGame רץ מ-main thread → setValue() מותר. repository.saveGameResult() מנהל thread בעצמו.

public class FirebaseSyncWorker extends Worker {
    @Override @NonNull
    public Result doWork() {
        // רץ ב-background thread שכבר ניהל WorkManager
        List<GameEntity> unsynced = gameDao.getUnsynced();
        for (GameEntity g : unsynced) {
            boolean success = firebaseManager.uploadGame(g);
            if (success) {
                g.syncedToFirebase = true;
                gameDao.updateGame(g);
            }
        }
        return Result.success();
    }
}

8️⃣ StatisticsViewModel – postValue מ-Thread▼
public class StatisticsViewModel extends ViewModel {
    private final MutableLiveData<StatisticsData> statisticsData = new MutableLiveData<>();
    private final MutableLiveData<Boolean> isLoading = new MutableLiveData<>(false);

    public void loadStatistics() {
        isLoading.setValue(true); // main thread — setValue OK
        new Thread(() -> {
            List<Game> allGames = gameRepository.getAllGames();
            StatisticsData stats = calculateStatistics(allGames);
            statisticsData.postValue(stats); // background thread — חייב postValue!
            isLoading.postValue(false);
        }).start();
    }

    private StatisticsData calculateStatistics(List<Game> games) {
        int gamesWon = 0, bestStreak = 0, tempStreak = 0;
        for (Game game : games) {
            if (game.getStatus() == GameStatus.WON) {
                gamesWon++; tempStreak++;
                if (tempStreak > bestStreak) bestStreak = tempStreak;
            } else { tempStreak = 0; }
        }
        return new StatisticsData(gamesWon, games.size(), bestStreak);
    }
}
חשוב: setValue() מ-background thread → IllegalStateException. postValue() thread-safe תמיד.
9️⃣ AES Encryption – EncryptionUtil▼
הרחבה — הצפנה. AES = סימטרי: אותו key להצפנה ולפענוח.
public static String encrypt(String data, String key) throws Exception {
    SecretKeySpec secretKey = new SecretKeySpec(key.getBytes(), "AES");
    Cipher cipher = Cipher.getInstance("AES");
    cipher.init(Cipher.ENCRYPT_MODE, secretKey);
    return Base64.encodeToString(cipher.doFinal(data.getBytes()), Base64.DEFAULT);
}

public static String decrypt(String encrypted, String key) throws Exception {
    SecretKeySpec secretKey = new SecretKeySpec(key.getBytes(), "AES");
    Cipher cipher = Cipher.getInstance("AES");
    cipher.init(Cipher.DECRYPT_MODE, secretKey);
    return new String(cipher.doFinal(Base64.decode(encrypted, Base64.DEFAULT)));
}
שימוש: מצפין boardState לפני שמירה ב-Firestore. Base64 ממיר bytes → String לשמירה כ-text.

// Score Calculation — GameEngine.calculateScore()
public int calculateScore(int seconds, int moves, int difficulty, int flagsPlaced, int totalMines) {
    int timeBonus         = Math.max(1000 - seconds, 0);
    int moveBonus         = Math.max(500 - moves, 0);
    int difficultyBonus   = difficulty * 10;           // minesCount * 10
    int flagAccuracyBonus = (flagsPlaced == totalMines) ? 200
                          : Math.max(0, 100 - Math.abs(flagsPlaced - totalMines) * 10);
    return timeBonus + moveBonus + difficultyBonus + flagAccuracyBonus;
}
